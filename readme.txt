Displays youtube channel subscribe button using channel-name or channel-id in Short Code.

== Description ==

Displays youtube channel subscribe button using channel-name or channel-id in Short Code.
Short Code : 
[youtube_subscribe]
[youtube_subscribe channel=yourChannelName]
[youtube_subscribe channel_id=yourChannelId]

Find your youtube id https://www.youtube.com/account_advanced

= Requirements =

* Latest version of WordPress

= Feature =

* You can use these shortcodes multiple times with different channel-name or channel-id.
* Admin side shortcode page.
* You can place this shortcode either in post or page

*Youtube subscribe button*

Create a new post/page and use the following shortcode to create a Youtube subscribe button:

`[youtube_subscribe channel=subarumydrug]`
`[youtube_subscribe channel_id=subarumydrug]`

== Installation ==

1. Go to the Add New plugins screen in your WordPress Dashboard
1. Click the upload tab
1. Browse for the plugin file(youtube-subscribe.zip) on your computer
1. Click "Install Now" and then hit the activate button
